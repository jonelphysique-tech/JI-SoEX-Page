What JI-SoEX Tools Does

Every module writes straight into the active STAAD.Pro model through OpenSTAAD. Open your model, click Connect, and work through the tabs in order — or jump to just the one you need.

Frame
TAB · FR

Pick a structural template, set X/Z bays and Y levels, and push a complete geometry grid into the active model.

Materials
TAB · MATS

Concrete constants and ready-made material sets — 21 MPa, 28 MPa, dummy, slab, and shearwall.

Properties
TAB · PROPS

Column, beam, and tie-beam sections with member rotation, beam-end display, and cracked-section factors.

Specifications
TAB · SPEC

Member releases, VBS-style auto offsets, slabs, and rigid floor diaphragms applied across the model.

Supports
TAB · SU

Fixed or pinned support assignment at base level, driven from the generated node grid.

Groups
TAB · GR

Floor-by-floor member and node groups, created automatically from the model's level structure.

Load Combinations
TAB · LOADS

Basic, school-style, and full NSCP 2015 generators. Auto-detects DL, LL, LLR/RL, EX/EZ, RSX/RSZ, and WX/WZ titles.

Load Generation
TAB · LOAD GEN

Generate and apply load patterns onto the model, ready to be rolled into the combination generator.

Response Spectra
TAB · RS

UBC 97 / NSCP 2015 spectrum curve with control periods, base-shear checks, and copyable spectrum points.

Footing Design
TAB · DE

Isolated footing design to NSCP 2015 / ACI 318-14 — biaxial pressure, punching, one-way shear, flexure, stability, and PDF report.

Select Tools
TOOL WINDOW

Node and member selection helpers with target, tolerance, and Y-level filters, synced to the STAAD selection.

Run Analysis
TAB · RUN

Submit the STAAD.Pro analysis without leaving the app, then extract reactions straight into footing design.

License Panel
STATUS BAR

Frequently Asked
Do I need STAAD.Pro to use it?

Yes. JI-SoEX Tools is an automation layer, not an analysis engine — it drives your own licensed STAAD.Pro installation through the OpenSTAAD COM bridge.

How does activation work?

Licenses are offline and tied to one registered PC. Install the app, copy the PC ID from the License panel, send it to us, and paste back the activation key you receive.

Can I move my license to another PC?

A license is issued against a single registered PC ID. If you change machines, message us with the new PC ID and we will sort out a transfer.

Why does an action say a method is unavailable?

The automation layer uses OpenSTAAD write methods where your installed STAAD.Pro exposes them. If one is missing, the app names the exact operation so it can be mapped to your STAAD.Pro version.

Is the download safe to run?

Verify it before installing — the SHA-256 of the installer is published above and in the release. Windows SmartScreen may warn on first run because the installer is new; choose More info → Run anyway once you have checked the hash.

Ready to automate your STAAD workflow?

Download the installer now and try it against your own model. Message us with your PC ID when you are ready for a license key — perpetual access is a one-time PHP 3,000.

How JI-SoEX Will WorkBROWSER API

JI-SoEX is not a standalone analysis tool — it is a result extraction and design layer that sits on top of your existing structural analysis software. It reads model geometry, load cases, and computed member forces through a local browser API, then performs full RC member design, P-M interaction checks, and envelope visualization without re-entering a single value.
