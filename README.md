# JI-SoEX | Structural Result Extractor & Design Suite

**JI-SoEX** is an automated structural engineering utility developed by **JRB iStruct Engineering Consultancy**. It is designed to bridge the gap between heavy structural analysis software and immediate, code-compliant reinforced concrete design.

By utilizing a secure API bridge, JI-SoEX extracts member forces directly from your analysis models and instantly generates design checks and reports based on **NSCP 2015** and **ACI 318-14** standards.

## 🚀 Key Features

* **Direct API Extraction:** Pulls member forces and analysis data without manual data entry.
* **Automated RC Design:** Instant code-compliant checks for Columns, Beams, and Footings.
* **Browser-Based Engine:** Fast, responsive local output engine for viewing and exporting results.
* **Hardware-Locked Security:** Secure, cloud-verified licensing system ensuring dedicated access for licensed engineers.

## 📥 How to Download and Install

This repository serves as the official distribution hub for the JI-SoEX compiled executables. **The proprietary Python source code is closed-source and is not hosted here.**

1. Navigate to the [Releases](../../releases) section on the right side of this repository.
2. Click on the latest version tag (e.g., `v2.0.0`).
3. Download the attached `JI-SoEX.exe` installer file under the **Assets** dropdown.
4. Run the application. Upon first launch, the software will generate a unique **PC Number**.
5. Contact JRB iStruct to register your PC Number and activate your license.
